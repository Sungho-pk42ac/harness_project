# 목록 정렬 토글 기능 정의서

## 기능 개요

노트 목록을 원하는 기준·방향으로 정렬해서 볼 수 있다.

## 기능 요구사항

- 노트 목록을 **수정일 / 생성일 / 제목** 기준으로 정렬할 수 있다
- **오름차순 / 내림차순** 방향을 바꿀 수 있다
- 정렬 컨트롤(드롭다운)을 노트 목록 상단에 둔다

## 비고

- 지금 목록은 `db.json`이 준 순서 그대로라 노트가 쌓이면 원하는 걸 못 찾는다.
- 검색(필터)과 역할이 다르다 — 검색은 "거르기", 정렬은 "순서 바꾸기".
- 서버 변경 없이 `NoteList`의 정렬 로직만으로 체감 편의가 크다.
- 사용자 수동(드래그) 순서·서버 정렬은 범위 밖.
