# 로그인 기능 정의서

## 기능 개요

사용자가 로그인하여 자신의 노트에 접근할 수 있다.

## 기능 요구사항

- 사용자가 아이디(이메일)와 비밀번호로 로그인할 수 있다
- 로그인한 사용자는 로그아웃할 수 있다
- 로그인 상태가 유지되어, 새로고침해도 다시 로그인하지 않아도 된다
- 로그인하지 않은 사용자는 노트 화면에 접근할 수 없다

## 비고

- 현재 백엔드는 JSON Server(`db.json`)로, 실제 인증 서버가 없다 (구현 방식은 이후 결정)
- 비밀번호 저장/검증 방식, 회원가입 포함 여부 등은 ②단계에서 확정
