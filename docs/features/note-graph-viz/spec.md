# note-graph-viz — 초안 정의서 (spec)

## 개요

노트를 **노드**, 노트 사이 관계를 **엣지**로 그려 브라우저에서 보는 **노트 관계 그래프 시각화**.
Neo4j(<https://github.com/neo4j/neo4j>)의 **속성 그래프 모델**(노드+관계+속성)을 개념적으로 참고하되,
**별도 그래프 DB 서버는 두지 않고** 기존 Supabase 노트 데이터에서 그래프를 만들어 클라이언트에서 렌더한다.

## 핵심 요구사항 (러프)

- 활성 노트(휴지통 제외)를 노드로 표시. 노드 = 노트 제목.
- 엣지 = 노트 간 관계. MVP는 **공유 태그**(두 노트가 같은 태그를 가지면 연결).
- 노드 클릭 → 해당 노트 선택/열기.
- 헤더 화면 토글에 graph 뷰 추가(notes / trash / **graph**).
- 노트가 0~1개 등 그릴 게 없으면 친화 빈 상태.

## 비고 (제약)

- 그래프 빌드(노트[]→{nodes,edges})·레이아웃 계산은 **순수 함수**(`src/utils`/`src/lib`)로 두고 단위 테스트.
- 렌더는 디자인 시스템 준수: 임의 hex/그림자/인라인 style 금지 → SVG에 Tailwind 시맨틱 토큰 클래스.
- 물리 시뮬레이션·줌/팬·드래그·실시간은 MVP 제외(후속).
- 별도 그래프 DB·신규 무거운 라이브러리 도입은 지양(의존성 없는 SVG 우선 검토).
