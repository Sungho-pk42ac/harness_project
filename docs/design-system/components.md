# 컴포넌트 인벤토리

디자인 캡쳐에서 관찰된 컴포넌트 목록입니다. 각 항목은 **용도 / 구성 / 토큰 사용**을 간략히 정리했습니다.
구현 시 [colors.md](colors.md)의 토큰과 [typography.md](typography.md)의 스케일을 따릅니다.

## 전역 / 레이아웃

### 상단 네비게이션 바

- **용도**: 모든 페이지 최상단 고정. 브랜드와 전역 액션.
- **구성**: 좌측 로고("SLNOTE", `font-display`) / 중앙·우측 메뉴 링크 / 우측 아이콘(검색·알림·프로필) + 우측 끝 강조 버튼.
- **토큰**: 배경 `card`(흰색), 하단 `border`, 링크 `foreground`/`muted-foreground`, 강조 버튼 `primary`.

### 다단 문서 레이아웃

- **용도**: 문서 상세 페이지. 좌측 목차 / 중앙 본문 / 우측 아웃라인의 3단.
- **구성**: 좌 TOC 사이드바 + 중앙 본문(읽기 폭 제한) + 우 보조 아웃라인. 좁은 화면에선 사이드바 접힘.
- **토큰**: 본문 영역 `card`, 사이드바 텍스트 `muted-foreground`, 현재 항목 강조 `primary`.

## 콘텐츠

### 히어로 섹션

- **용도**: 랜딩 최상단. 서비스 한 줄 가치 제안 + 진입 액션.
- **구성**: 상단 작은 배지 → 큰 디스플레이 헤딩(한글 Bold) → 서브텍스트 → CTA 버튼들.
- **토큰**: 헤딩 `foreground`, 서브 `muted-foreground`, 주 버튼 `primary`.

### 콘텐츠 카드

- **용도**: 노트/문서 한 건을 그리드로 나열.
- **구성**: 썸네일 이미지(상단) → 제목 → 설명(2줄 클램프) → 하단 작성자 아바타 + 이름 + 날짜.
- **토큰**: 면 `card`, 테두리 `border`, 제목 `foreground`, 메타 `muted-foreground`, 일관된 radius·그림자 하나.

### 가로 추천 카드 행

- **용도**: 상단 추천/주요 콘텐츠를 가로 스크롤 또는 한 줄로.
- **구성**: 어두운 썸네일 위주의 카드 4~5개를 가로 정렬.

## 요소

### 버튼

- **Primary**: `bg-primary text-primary-foreground`, hover `primary-hover`. 핵심 액션 1개에만.
- **Secondary(아웃라인)**: 투명/`muted` 배경 + `border`, 텍스트 `foreground`. 보조 액션.
- 둘 다 `rounded-xl`, 동일 높이/패딩.

### 배지 / 필 칩

- **용도**: 카테고리, NEW 라벨, 태그.
- **토큰**: 연한 `primary-subtle` 배경 + `primary` 텍스트, 또는 중립 `muted` + `muted-foreground`. `rounded-full`, `text-xs`.

### 아바타

- **용도**: 작성자 표시.
- **구성**: 원형(`rounded-full`) 이미지, 카드 하단·댓글 등에서 작게.

## 오버레이 / 플로팅

### 검색 모달

- **용도**: 전역 검색. 단축키/검색 아이콘으로 진입.
- **구성**: 화면 중앙 카드형 모달 + 뒤 배경 딤(어둡게). 상단 검색 입력 + 결과/안내.
- **토큰**: 모달 `card`, 딤은 반투명 검정 오버레이, 포커스 링 `ring`.

### 플로팅 액션 버튼 (FAB)

- **용도**: 우하단 고정. 빠른 새 노트/주요 액션.
- **구성**: 원형(`rounded-full`) 인디고 버튼 + 아이콘, 살짝 떠 보이는 그림자.
- **토큰**: `bg-primary text-primary-foreground`, hover `primary-hover`.
